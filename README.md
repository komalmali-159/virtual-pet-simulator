# Virtual Pet Simulator

## Overview
This is a web-based Virtual Pet Simulator created using HTML, CSS, and JavaScript.
The simulator allows users to feed, play with, and put a virtual pet to sleep.
The petâ€™s hunger, happiness, and energy change over time and are displayed using progress bars.
The pet also shows different states such as happy, sad, and sleeping.

This project was developed as part of internship and web development practice.

---

## How to Run the Project
1. Download or copy all files into a single folder:
   - index.html
   - styles.css
   - script.js
2. Open `index.html` in any modern web browser (Chrome, Edge, or Firefox).
